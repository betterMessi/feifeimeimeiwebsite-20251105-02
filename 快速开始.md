# 🚀 快速开始 - 5分钟上手

## 第一步：安装依赖

```bash
cd backend
npm install
```

## 第二步：启动后端服务器

```bash
npm start
```

你会看到：
```
✅ 数据库初始化完成
✅ 已创建默认用户: user1/password1, user2/password2
🚀 服务器运行在 http://localhost:3000
```

## 第三步：初始化标签（可选）

在另一个终端窗口：

```bash
cd backend
node init-tags.js
```

这会创建默认标签：旅行、美食、日常、风景、节日、自然

## 第四步：打开前端

1. 直接双击打开 `样式预览/index.html`
2. 或者使用本地服务器：
   ```bash
   cd 样式预览
   python -m http.server 8080
   # 然后访问 http://localhost:8080
   ```

## 第五步：开始使用！

### 上传照片/视频
1. 点击"上传"菜单
2. 拖拽文件到上传区域，或点击"选择文件"
3. 选择标签（可选）
4. 点击上传

### 查看照片
- **时间线**：按时间顺序查看所有照片
- **分类**：按标签分类查看

## 📝 注意事项

1. **API地址**：确保 `样式预览/script.js` 中的 `API_BASE_URL` 是 `http://localhost:3000/api`

2. **文件大小**：单文件最大 50MB

3. **支持格式**：
   - 图片：JPG, PNG, GIF, WebP
   - 视频：MP4, WebM, MOV

4. **数据库**：首次运行会自动创建 `backend/database.db`

## 🌐 部署到公网

### 最简单的方式：Railway

1. 访问 https://railway.app
2. 注册账号（GitHub登录）
3. 创建新项目
4. 连接GitHub仓库
5. 选择 `backend` 目录
6. 自动部署！

详细步骤见 `部署指南.md`

## ❓ 遇到问题？

### 后端无法启动
- 检查是否已安装 Node.js 18+
- 检查端口 3000 是否被占用

### 上传失败
- 确保后端服务器正在运行
- 检查文件大小和格式

### 图片无法显示
- 检查后端是否运行在 `http://localhost:3000`
- 检查浏览器控制台是否有错误
