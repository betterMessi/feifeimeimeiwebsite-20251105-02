# 清空数据说明

## 如何清空所有上传内容和数据库

### 方法一：使用清空脚本（推荐）

1. **停止后端服务器**（如果正在运行）

2. **运行清空脚本**：
   ```bash
   cd backend
   node clear-data.js
   ```

3. **重新初始化数据库**：
   ```bash
   node init-tags.js
   ```

4. **重新启动服务器**：
   ```bash
   npm start
   ```

### 方法二：手动删除

1. **删除上传文件**：
   - 删除 `backend/uploads` 文件夹中的所有文件（保留文件夹结构）

2. **删除数据库文件**：
   - 删除 `backend/database.db` 文件（如果存在）

3. **重新初始化**：
   ```bash
   cd backend
   node init-tags.js
   ```

## 注意事项

⚠️ **警告**：清空操作会永久删除所有上传的照片、视频和数据库记录，此操作不可恢复！

✅ 清空后，系统会自动创建新的数据库并插入默认标签：
- 肥肥美美
- 狗娃儿之家
- 证件照
- 旅游
- 吃吃喝喝
- 花花
- 公主的眼影
